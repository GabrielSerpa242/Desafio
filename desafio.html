Oque é HTML?

(Voltar)

HTML, ou HyperText Markup Language (linguagem de Marcação de hipertexto), é uma linguagem de marcação usada para criar a estrutura de páginas web. É a base de qualquer página da web, definindo como o conteúdo, como texto, imagens e links, é organizado e exibido em um navegador. HTML não é uma linguagem de programação, mas sim uma linguagem que descreve a estrutura e o conteúdo da web.

  • Adicione pelo menos 5 linhas de dados fictícios.
  • Estilize a tabela para que o cabeçalho tenha fundo preto e texto branco.
  • As linhas pares devem ter um fundo cinza claro.
  • Ao passar o mouse sobre uma linha, ela deve mudar de cor.

Oque HTML faz?

(Voltar)
  • Estrutura a página: Define como o conteúdo é organizado (parágrafos, títulos, listas, etc.);
  • Apresenta elementos visuais: Permite a inserção de imagens, vídeos e outros elementos;
  • Cria links: Permite a criação de hyperlinks para conectar diferentes páginas e sites.

Como ele funciona?
